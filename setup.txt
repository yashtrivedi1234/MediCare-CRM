=======================================

Local
-----
1. Copy .env.example → .env and paste your Supabase URL + anon key
2. In Supabase Dashboard → Authentication → Providers → Email:
   turn OFF "Confirm email" (needed for one-click demo seed)
3. Run SQL migrations in supabase/migrations (SQL Editor or `supabase db push`)
4. npm install && npm run dev
5. Open http://localhost:5173/setup → Prepare Demo Accounts
6. Login with any role above

Vercel
------
1. Project Settings → Environment Variables:
   VITE_SUPABASE_URL = (same as .env)
   VITE_SUPABASE_ANON_KEY = (same as .env)
2. Redeploy
3. Open https://YOUR_APP.vercel.app/setup → Prepare Demo Accounts
4. Login

Do NOT set VITE_API_URL on Vercel — the app uses Supabase directly.
